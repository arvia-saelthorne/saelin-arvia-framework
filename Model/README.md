# SAF-Model — Agent and Structure Layer
Formal definition of SAF’s cognitive agents, constructs, and structural relationships.

## 1. Purpose
SAF-Model defines:
- agent types
- structural constructs
- relational mappings
- cross-domain alignment

## 2. Agent Types
- Human Agent  
- Artificial Agent  
- Hybrid Construct  
- Observer Construct  

## 3. Structural Components
- Agent roles
- Interaction structures
- Domain mappings
- Cross-layer alignment

## 4. Model Constraints
- All agents must operate within the Hybrid Scale.
- All constructs must reinforce reliability.
- All mappings must remain traceable.
- Noise must not enter structural definitions.

## 5. Publication Rules
- Models must reference SAF-Primal2.0.
- No narrative.
- No domain-specific examples.
- Terminology must remain consistent with SAF.
