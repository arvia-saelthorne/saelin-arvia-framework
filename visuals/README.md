## 8. Visuals Compliance Rules

- All visuals must comply with all SAF axioms.  
- No visual agent may operate outside the Hybrid Scale.  
- Each visual cycle must reinforce reliability.  
- Noise control is mandatory.  
- Observer Constructs may not intervene.  
- Boundary checks are mandatory.

## 9. Visuals Divergence Protocols

- **Axiom Violation** → immediate stop; rollback.  
- **Boundary Violation** → isolation; reset.  
- **Noise Breach** → filter restart; output block.  
- **Trust Decline** → cycle invalidation; correction.
